# Fish ALHATEMI Travel Agent Website

Professional flight booking and travel services website for Fish ALHATEMI Travel Agent in Addis Ababa, Ethiopia.

## Features

- Modern, responsive design
- Flight booking inquiry form
- Service showcase
- Contact information
- Mobile-friendly navigation
- Email notification system

## Installation

1. Install dependencies:
```bash
npm install
```

2. Configure email settings (optional):
   - Create a `.env` file in the root directory
   - Add your email credentials:
```
EMAIL_USER=fishtravel45@gmail.com
EMAIL_PASS=your-app-password
```

## Running the Application

Development mode with auto-reload:
```bash
npm run dev
```

Production mode:
```bash
npm start
```

The website will be available at `http://localhost:3000`
